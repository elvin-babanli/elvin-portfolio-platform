# Elvin Portfolio Platform

Personal developer portfolio platform designed to present projects, technical experience, and ongoing development work.

Live platform:  
https://elvin-babanli.com

---

## Overview

This project is a production-deployed portfolio platform built to showcase software projects, backend development experience, and technical capabilities.

The platform allows visitors and recruiters to:

- explore software projects visually
- view technical background and experience
- interact with an AI assistant trained to answer questions about my work
- navigate live project demos and GitHub repositories

The project is actively maintained and continuously updated.

---

## Key Features

• Production-deployed personal developer platform  
• AI-powered recruiter assistant using OpenAI API  
• Interactive project showcase  
• Secure backend architecture  
• Domain connection and deployment  
• Recruiter-friendly interface for exploring projects  

---

## AI Recruiter Assistant

The platform includes an AI assistant integrated with OpenAI APIs.

It can answer questions such as:

- What technologies does Elvin use?
- What backend projects has he built?
- What experience does he have with APIs?
- What tools and frameworks does he work with?

This allows recruiters to quickly explore technical details through an interactive interface.
